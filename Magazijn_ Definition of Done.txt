Definition of Done


* Code compileert zonder errors.
* Code is geschreven met behouding van clean code afspraken 
* Controller (Endpoint) tests zijn geslaagd
* Peer review is gedaan door iemand die de code niet zelf heeft geschreven (PO of Scrum master)
* Code is refactored (indien mogelijk)
* Main branch heeft altijd werkende code
* Nieuwe functionaliteit is goedgekeurd door PO
* Communiceer met het team wanneer je met main wil mergen
* Een sprint task is done wanneer de code gemerged is met de main branch
Afspraken Clean Code
* Lower Camel Case (variabelen, functies)
* Upper Camel Case (classes, interfaces, enums enz)
* Beschrijvende benaming voor functies, variabelen, classes enz
* Intellij de code laten herformatteren
* Member variabelen hebben dezelfde naam zoals in de database
